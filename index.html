WorldCup26 publico

Activo por defecto. Consulta partidos, grupos, equipos y marcadores en linea sin API key.

window.WORLD_CUP_CONFIG = { provider: "worldcup26", endpointBase: "https://worldcup26.ir", autoRefreshMinutes: 2 };

API-FOOTBALL / API-SPORTS

Usa fixtures, standings, livescore y estadisticas cuando tengas API key.

window.WORLD_CUP_CONFIG = { provider: "api-football", apiKey: "TU_API_KEY", season: 2026, leagueId: 1 };

football-data.org

Alternativa sencilla para partidos, tablas y marcadores en formato machine-readable.

window.WORLD_CUP_CONFIG = { provider: "football-data", apiKey: "TU_API_KEY", competition: "WC" };

Sportmonks

Alternativa premium para fixtures, live scores, grupos y bracket.

window.WORLD_CUP_CONFIG = { provider: "sportmonks", apiKey: "TU_API_KEY", seasonId: "WORLD_CUP_2026" };

Modo local

La pagina intenta actualizar en linea cada 2 minutos. Si la API falla, usa data/worldcup.json; si el navegador no permite leer JSON por doble clic, usa una copia interna para que nunca se rompa.
